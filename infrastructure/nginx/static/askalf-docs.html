Quick Start

Deploy AskAlf in 60 seconds:

git clone https://github.com/SprayberryLabs/askalf.git
cd askalf && ./setup.sh
docker compose -f docker-compose.selfhosted.yml up -d

Open http://localhost:3001 and complete the 5-step onboarding wizard.

Requirements

  • Docker and Docker Compose
  • 4GB+ RAM (8GB recommended)
  • At least one AI provider key (Anthropic recommended)
  • Git

Agent Fleet

AskAlf ships with 7 specialized agents:

  • Backend Dev — API routes, database queries, server-side logic
  • Frontend Dev — React components, UI features, styling
  • QA — Tests, validation, quality assurance
  • Infra — Docker, deployments, infrastructure management
  • Security — Vulnerability scanning, dependency audits, secret detection
  • Watchdog — Health checks, incident detection, monitoring
  • Writer — Documentation, changelogs, technical writing

Onboarding Wizard

The 5-step setup guides you through:

  1. Workspace — Name your deployment
  2. AI Config — Choose basic or enhanced intent parser, optionally add OpenAI key for embeddings
  3. Connect Claude — OAuth flow for agent execution (or skip)
  4. Appearance — Dark, light, or system theme
  5. Launch — Review and enter the command center
