Quick Start

Running in under 2 minutes

~/askalfbash
# Clone, configure, run
$ git clone https://github.com/SprayberryLabs/askalf.git && cd askalf
$ ./setup.sh          # generates secrets, creates .env
$ nano .env            # add your Anthropic/OpenAI key
$ docker compose up -d
