Quick Start

Running in under 2 minutes

~/askalfbash
# Clone, configure, run
$ git clone https://github.com/SprayberryLabs/askalf.git && cd askalf
$ cp .env.example .env     # add your Anthropic or OpenAI key
$ docker compose -f docker-compose.selfhosted.yml up -d
