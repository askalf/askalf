THE BRAIN

1,500 nodes of knowledge.
Built overnight. Without instruction.

The agents don't just execute — they learn. Every fix, every deployment, every failure is encoded into a 10-layer cognitive memory system that grows on its own.

Semantic Memory

Facts, concepts, relationships. "Redis is a cache layer." "The auth middleware checks JWT tokens." Cross-linked into a searchable knowledge graph.

Episodic Memory

What happened and what worked. "Last time the DB locked up, increasing work_mem fixed it." Reinforced by outcomes — successes strengthen, failures decay.

Procedural Memory

How to do things. Patterns, workflows, rules. Extracted from successful executions and consolidated into reusable procedures.

The wildest part: nobody told the agents to build this graph. They discovered 1,500 relationships between concepts, services, and patterns on their own. That's not automation. That's emergent intelligence.

QUESTIONS YOU'RE THINKING

Straight answers.


Is this vaporware?

No. It's running right now on real infrastructure. The agents that built this platform are the same agents that ship to you. The knowledge graph, the memory system, the orchestration — all production code.

How much does it cost to run?

Infrastructure: ~$5/month (small VPS + Docker). AI costs: depends on usage. Our fleet runs 12 executions overnight for ~$0.43. Cost guardrails prevent runaway spending.

Do I need all 7 agents?

No. Disable any agent you don't need. Create custom agents with the Builder. Each agent is independently configurable with its own model, budget, and schedule.

What AI providers does it support?

Anthropic (Claude) for agent execution via OAuth. OpenAI for embeddings. Google AI optional. The intent parser works with Anthropic or in basic keyword mode with no API key.

Can the agents modify production code?

Yes — that's the point. They create branches, commit code, and deploy. You set the autonomy level. Full auto, human-in-the-loop approval, or manual dispatch only.

What happens if an agent breaks something?

Watchdog detects it. QA validates it. Infra rolls it back. The ticket system creates a paper trail. Cost guardrails cap spending. You get a morning briefing of everything that happened.

Is my data safe?

Your data never leaves your infrastructure. No telemetry, no analytics, no phone-home. Everything runs in your Docker containers on your hardware. Open source — audit every line.

How is this different from ChatGPT/Claude?

Those are one mind answering one question. This is seven specialists who coordinate, delegate work to each other, share memory, learn from outcomes, and operate 24/7 without prompting.
