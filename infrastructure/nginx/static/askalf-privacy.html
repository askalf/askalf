Privacy Policy

Last updated: March 14, 2026

The Short Version

AskAlf is self-hosted software. Your data stays on your infrastructure. We don't collect, store, or transmit any of your data.

Self-Hosted Deployment

AskAlf runs entirely on your own hardware. All data — agent executions, memory, configurations, credentials, and conversations — is stored in your PostgreSQL database and Redis instance. Nothing leaves your network unless you explicitly configure external integrations.

What We Don't Collect

  • We don't collect usage analytics or telemetry
  • We don't track user behavior
  • We don't store your API keys (they stay in your environment)
  • We don't have access to your agent executions or outputs
  • We don't phone home or send crash reports

External Services

When you use AskAlf, the following external services may be contacted based on your configuration:

  • AI Providers (Anthropic, OpenAI, Google) — for agent execution and embeddings, using your API keys
  • OAuth (Anthropic) — for Claude CLI authentication, if configured
  • Integrations (GitHub, Slack, etc.) — only the ones you explicitly connect
  • Font APIs (Google Fonts, FontShare) — for dashboard typography

askalf.org Website

The askalf.org marketing website is a static HTML page. It does not use cookies, analytics, or tracking scripts. No personal data is collected from visitors.

Open Source

AskAlf is open-source software. You can audit the entire codebase at github.com/SprayberryLabs/askalf.

Contact

Questions about privacy? Open an issue on GitHub or reach out via the repository.
