Architecture

Five layers. Zero blind spots.

A complete system for recording, branching, replaying, analyzing, and sharing AI conversation timelines.

Layer 01

Timeline Engine

A Merkle DAG — like Git, but for conversations. Every message is a content-addressed node with parent pointers. Branch, merge, and diff any two points in a conversation tree with cryptographic integrity.

Every node hash includes its parent hash.
Tamper-proof. Deduplicated. Instant diffs.
Layer 02

State Snapshots

Copy-on-write context reconstruction. Instead of storing full snapshots, we store deltas — exactly what changed at each message. Walk backward to rebuild any point-in-time state in milliseconds.

Base snapshot + deltas = ZFS for conversations.
Stores the exact API request payload hash.
Layer 03

Ghost Mode

Speculative execution for conversations. Before you send a message, Ghost Mode projects multiple possible futures using lightweight preview models. See where each path leads before committing.

CPU speculative execution, applied to LLMs.
Divergence scoring ranks which futures matter.
Layer 04

Forensics Engine

Post-hoc conversation autopsy. Context pressure curves, confidence decay signals, tool failure cascades, semantic drift detection. See exactly where and why a conversation degraded.

Chrome DevTools Performance tab,
but for AI conversations.
Layer 05

Sharing & Déjà Vu

Export conversations as portable .claw capsules. Import others' timelines as read-only branches you can fork. Déjà Vu detection warns when you're heading down a path that failed before.

npm for conversation timelines.
Community-powered failure prevention.
Workflow

Five steps. Total visibility.

Record

Every message becomes a Merkle node. Content-addressed, immutable, with full execution metadata.

Branch

Fork at any point. Explore alternative prompts, different models, varied context. Each branch preserves full lineage.

Replay

Reconstruct the exact model context at any message. See precisely what the model saw — the effective prompt, the full payload.

Analyze

Forensics engine traces context pressure, confidence decay, and semantic drift. Pinpoint the exact message where degradation began.

Share

Export as a .claw file. One portable capsule containing the full timeline. Import, fork, and learn from others' sessions.

Component Technology Rationale
