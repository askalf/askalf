MCP Configuration

Add this to your Claude Desktop config to connect to SUBSTRATE. Replace <your-api-key> with an API key from Dashboard → API Keys.

Config location: %APPDATA%\Claude\claude_desktop_config.json (Windows) or ~/Library/Application Support/Claude/claude_desktop_config.json (macOS)

{ "mcpServers": { "substrate": { "command": "npx", "args": [ "-y", "mcp-remote", "https://orcastr8r.com/mcp/sse", "--header", "Authorization:${AUTH_HEADER}" ], "env": { "AUTH_HEADER": "Bearer <your-api-key>" } } } }

Note: Uses mcp-remote to bridge Claude Desktop's stdio transport to our SSE server. Requires Node.js installed.
