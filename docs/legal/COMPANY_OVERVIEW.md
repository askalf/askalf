## Business Model

### Revenue Streams

1. **Subscription Plans**
   - Free: 5 daily credits, limited models
   - Basic ($5/mo): 50 credits, API key access
   - Pro ($15/mo): 200 credits, BYOK, all models
   - Team ($49/mo): 500 credits, workspace features
   - Enterprise (Custom): Unlimited, dedicated support

2. **BYOK (Bring Your Own Keys)**
   - Zero markup on API pass-through
   - Users benefit from shard caching even with own keys

3. **Future: API Access for Developers**
   - Metered API usage for B2B integrations

## Technology

### SUBSTRATE Architecture
A 4-tier cognitive memory system:
1. Procedural Memory (Shards) - Reusable response patterns
2. Episodic Memory (Traces) - Interaction history
3. Semantic Memory (Facts) - Pure knowledge
4. Working Memory - Active context

### Shard Technology
- Patent-pending pattern matching system
- Eliminates redundant LLM calls
- Sub-100ms response for cached patterns
- Continuous improvement through execution feedback

### Environmental Impact
Per 1000 LLM tokens avoided:
- ~500ml water saved (data center cooling)
- ~10 Wh electricity saved
- ~5g CO2 avoided
