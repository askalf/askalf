# create-askalf

### One command. No Docker.

```bash
npx create-askalf
```

Sets up the full AskAlf AI workforce platform in a single Node.js process. PGlite for the database, in-memory cache, zero external dependencies.

Part of [AskAlf](https://askalf.org) — the first self-healing AI workforce with a nervous system, immune system, and collective memory.

## What it does

1. Asks for your AI provider API key (Anthropic, OpenAI, or both)
2. Generates secure configuration
3. Downloads AskAlf
4. Starts the platform

## Quick mode

```bash
ANTHROPIC_API_KEY=sk-ant-... npx create-askalf --quick
```

Skips prompts, uses defaults.
